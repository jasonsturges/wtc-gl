      The parameters to initialise the renderer.

      • Optional alpha?: boolean

        Whether to render an alpha channel.

      • Optional antialias?: boolean
      • Optional autoClear?: boolean

        Whether to clear the canvas between draw.

      • Optional canvas?: HTMLCanvasElement

        The canvas HTML element to render to.

      • Optional depth?: boolean

        Whether to render a depth buffer.

      • Optional dpr?: number

        The pixel aspect ratio of the canvas.

      • Optional height?: number

        The height of the canvas.

      • Optional powerPreference?: string

        WebGL power preference.

      • Optional premultipliedAlpha?: boolean

        Whether to use premultiplied alpha.

      • Optional preserveDrawingBuffer?: boolean

        Preserve the drawing buffer between calls. Useful for when downloading the canvas to an image.

      • Optional stencil?: boolean

        Whether to render a stencil buffer.

      • Optional webgl?: number

        The webGL version to try to use - 1, or 2

      • Optional width?: number
